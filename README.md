### 2.2. 핵심 게임 플레이 (Core Gameplay)

- **플레이어 목표:** 마지막까지 살아남아 최종 보스를 클리어 하는 것
- **핵심 시스템 및 규칙**
  - 사방에서 몰려오는 웨이브를 모두 정리해 라운드를 클리어 하는 것
  - 웨이브 종료시 보상시 포탈 밑에 O O O 아이템 혹은 능력 3개 띄우고 그 중 1개 선택 가능 (새로운 능력 획득 가능 및 HP 회복 고정)
  - 기본공격은 마우스 방향에 따라, 그리고 기본공격을 강화할 수 있음
  - 1~3 Round 초원- 4~6 Round 독맵, 7~9 Round 구름 위, 10 Round 보스전
  - 각 라운드는 새벽/낮/밤 으로 구분
  - 보스 스테이지는 항상 고정 (뭐 어두컴컴한 사각형 지하 감옥, 겉에는 용암흐름)
  - 시작할 때 무기선택 or 캐릭터 선택으로 기본무기 픽스
  - 무기(능력)강화는 분야 나눠서 할 건지?(뭐 개수라던가 속성이라던가 등)
      (ex) 1증강: 공격속도 /// 2증강: 공격범위증가 - 근접: 원뿔커짐/원거리: 투사체 개수 증가 /// 3증강: 추가데미지(속성X 이펙트까지만)
  - 맵 오브젝트 파괴시 일회용 아이템 드랍 (폭탄, 시간정지, 피 회복, 무적 등)
  - 일반 몬스터는 무조건 경험치만 떨굼 (오브젝트 구현이 어려우면 엘리트몹(체력↑,색이 다른 등)이 일회용 아이템 드랍?)
  - 레벨업 기존 능력 강화 가능 (이동속도, 기본 공격력, 추가체력 등)
  - 기본 HP 방식 ( 최대 10칸 or HP바(100 이상) )
  - 기본 능력 강화 (공용) / ex. 쿨타임, 이동속도, 데미지 등
- **조작 방법:** WASD, 마우스
